What's the game about?

Inspired by “Vampire Survivors”, which is a survival game with roguelite elements.

In this game you can level up by killing enemies and each time you level up you will be rewarded with a weapon or upgrade. The game gets progressively harder, so death is inevitable.

How long can you survive?

Why did I make this project?

I made this game for a school exam, for which I have to create a game. I also had to make game design documents for it and document others playtesting it.

I started with creating a FDD (functional design document) which contains everything about the game.

After the FDD, I created a TDD (technical design document) where I explain how I'm going to code everything.

During the exam I let others playtest the game, which gave me valuable feedback on what I could improve on.

I had 5 sprints in total (1 week per sprint) and used Trello to plan everything.

What did I learn?

Tilemap + Spritesheets

I've never used a tilemap before, but I knew that I needed it for this game. I also found the spritesheets that the original game used and used them for my game too.

There was one problem though, the creator of the spritesheets sucked at making spritesheets. The sprites weren't the same height and the background also wasn't transparent.

Another bad thing was that for objects, he would put a grass background on it. Which means that you can't put the background on a stone floor for example, without removing it manually. It was a lot of effort to remove the backgrounds and to make the sprites work, but I got it done in the end.

The map in the game is pretty big too, so I'm happy I learned how to use tilemaps instead of placing them manually.

There are so many more things I could add

While making the FDD I listed many things that I wanted to add, but knew I didn't have time for. This includes the permanent shop, upgrade system, boss fights, character selection, animations and more weapons/passives.

I have another game exam after this one, which means I won't have time to add these features. Nonetheless I'm very proud of what I've made so far, it looks really good!
